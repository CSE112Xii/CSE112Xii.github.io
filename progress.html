Front End Development

How It Works

Walkthrough

Home

This is the user's homepage. We are placed in the front page with the user profile, various links (feed, friend management, settings), and ad.

Posting an Update

By clicking on "My Feed" at the top left corner, a "Post an update" box will appear.

-

After the user types in his/her update, he/she can now post the update by clicking on "Publish".

-

Note: In this video demonstration, we are unable to publish the post since we are not connected to the backend yet.

3. Liking a Post

The user can like a post by clicking on the 'heart' icon. The red heart will appear and the number accumulates.

4. Commenting on a Post

The user can comment on a post by clicking on the 'speech bubble' icon. Submit the comment by pressing enter.

-``

After comment has been published.

Editing a Comment

The user can edit on a post by clicking on the 'note-pen' icon (the middle icon in the above the textbox). After the edit is complete, submit by pressing on enter.

Deleting a Comment

The user can delete a post by clicking on the 'trashcan' icon. The comment will be deleted off from the post.

Friend Management

The user can manage his/her friend by clicking on the 'friends' icon at the top right corner (the very left of the three icons). This will bring up the friend's management page.

Return to User Profile

The user can return back to home page by clicking on the 'user' icon at the top right corner (the middle of the three icons).

User Profile and Settings

The user can edit his/her first/last name, date of birth email address, and password by clicking on the 'tool' icon (the right of the user's profile picture). After the user is finished with his/her changes, click on save.
